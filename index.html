§ 01

Thesis — how I’m building in the agent era.

I.

The product is no longer the dashboard. The product is the agent — and the seams where it hands control back.

II.

Coordination is free now. What’s scarce is taste — knowing which of a thousand generated paths is the one to ship.

III.

Speed of iteration is a proxy for conviction. If you can’t rebuild it in a weekend, you probably shouldn’t defend it for a quarter.

IV.

Trust is the only moat left. Every opaque recommendation spends it; every legible one earns it back.

2025AI / SaaSSolo build

Signal — an agent for product feedback.

Problem

Product teams drown in Slack, GitHub, support. Themes surface weeks late — if at all.

Approach

An agent reads every channel, scores sentiment and urgency, clusters intent, drafts the epic.

Stack

Ingestion → embeddings → urgency model → theme clusters → epic generator.

Effect

Weekly roadmap review collapsed from days to a ten-minute scan.

2025AI / FintechSolo build

Prism — the single next move.

Problem

Self-directed investors face N possible actions. “What should I do?” has no honest answer.

Approach

Solver first. Language second. The model narrates what the math already decided.

Stack

Holdings → tax + market model → ranked actions → conversational follow-up.

Effect

One defensible recommendation — with reasoning a retail investor can read and argue with.
